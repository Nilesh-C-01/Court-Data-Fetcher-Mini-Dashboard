This tool accesses only publicly available court information.

How to Use Court Data Fetcher
Searching for Cases
  1. Select the Case Type from the dropdown
  2. Enter the Case Number (e.g., 1234, 5678)
  3. Choose the Filing Year
  4. Click Search Case button
Downloading Documents
  • Click on any PDF link in the case details
  • The document will be downloaded automatically
  • PDFs include orders, judgments, and notices
Viewing Search History
  • Use the "Search History" link in the navigation
  • View all previous searches and their results
  • Click on any case to view detailed information
Tips:
  • Case numbers should be entered without the year (e.g., enter "1234" not "1234/2023")
  • Make sure to select the correct filing year
  • If a case is not found, verify the case details on the court website
  • Some cases may have restricted access or may not be digitized
